# React + Vite
Para iniciar con el proyecto tiene que tener instalado Node.js

Ingresamos a la carpeta:

`cd SocialNetwork-UNAS`

Luego ejecutamos:

`npm install`

Una vez instaladas las dependencias, ejecuta el comando de desarrollo

`npm run dev`

Librerias:
AXIOS
  `npm install axios`
